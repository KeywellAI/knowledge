# Keywell UI Skills Repo

The `keywell-ui` skills repo is the source of truth for product and app development knowledge used when building Keywell Platform client apps.

**Repo:** https://github.com/KeywellAI/keywell-ui/tree/main/skills

---

## What It Contains

Skills, patterns, and conventions for building apps on the Keywell Platform — including domain models, data source guidance, component patterns, and AI-assisted development prompts.

## When to Use It

- Before speccing or building any new Keywell client app
- When looking for existing patterns or reusable conventions
- When working with AI tools (Claude Code, Cursor) to generate or scaffold Keywell app code — the skills here give the model the context it needs to produce correct, platform-consistent output
